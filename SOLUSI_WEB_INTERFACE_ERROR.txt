Langkah 1: Buka Terminal/PowerShell
   Cara buka:
   1. Tekan Windows + R
   2. Ketik: powershell
   3. Tekan Enter

Langkah 2: Masuk ke Folder Project
   Ketik di terminal:
   cd "D:\Praktikum DWBI\xampp\htdocs\nurani"

   Ketik di terminal:

   Atau jika ngrok di PATH:
   ngrok http 80
   Tekan Enter
