# 6.1 Identifikasi Biaya Implementasi

Berikut adalah identifikasi biaya implementasi sistem perbaikan Teaching Management System (TMS) di MTS Nurul Aiman Tanjungsari Sumedang:

## Tabel 6.1 Identifikasi Biaya Implementasi Sistem TMS

| No | Aktivitas | Biaya |
|----|-----------|-------|
| 1 | Kustomisasi dan pengembangan fitur sistem TMS (Laravel) | Rp 5.000.000 - 8.000.000 |
| 2 | Hosting VPS atau cloud untuk deployment sistem | Rp 1.500.000 - 3.000.000 |
| 3 | Pengadaan perangkat keras tambahan (jika diperlukan) | Rp 2.000.000 - 5.000.000 |
| 4 | Perawatan rutin sistem dan dukungan teknis (per tahun) | Rp 2.000.000 - 3.000.000 |

---

## Penjelasan Detail Biaya

### 1. Kustomisasi dan Pengembangan Fitur Sistem TMS (Laravel)
**Biaya: Rp 5.000.000 - 8.000.000**

**Cakupan:**
- Pengembangan fitur dashboard untuk Kepala Sekolah, Guru, dan Tenaga Usaha
- Implementasi sistem manajemen jadwal mengajar
- Pengembangan fitur manajemen materi pembelajaran
- Implementasi sistem kuis dan evaluasi
- Pengembangan fitur laporan dan statistik
- Integrasi sistem notifikasi email
- Kustomisasi UI/UX sesuai kebutuhan sekolah
- Testing dan debugging sistem

**Estimasi:**
- Development: 3-4 bulan
- Testing: 1 bulan
- Deployment: 1 minggu

---

### 2. Hosting VPS atau Cloud untuk Deployment Sistem
**Biaya: Rp 1.500.000 - 3.000.000**

**Cakupan:**
- VPS/Cloud hosting untuk server aplikasi
- Database hosting (MySQL/PostgreSQL)
- Domain name (jika diperlukan)
- SSL Certificate (HTTPS)
- Backup otomatis
- Monitoring dan uptime guarantee

**Opsi Hosting:**
- **Hosting Web Standar:** Rp 1.500.000 - 2.000.000/tahun
  - Shared hosting dengan cPanel
  - MySQL database include
  - SSL gratis
  - Backup manual

- **VPS (Virtual Private Server):** Rp 2.500.000 - 3.000.000/tahun
  - Dedicated resources
  - Full control server
  - Backup otomatis
  - Scalable

- **Cloud Hosting (Railway/Render):** Rp 0 - 1.500.000/tahun
  - Gratis untuk tier awal
  - Auto-scaling
  - Database include
  - Pay as you go

**Rekomendasi:** Mulai dengan hosting web standar, upgrade ke VPS jika traffic meningkat.

---

### 3. Pengadaan Perangkat Keras Tambahan (Jika Diperlukan)
**Biaya: Rp 2.000.000 - 5.000.000**

**Cakupan:**
- Server komputer (jika hosting on-premise)
- Laptop/PC untuk development dan maintenance
- Storage tambahan untuk backup
- Network equipment (router, switch) jika diperlukan
- UPS (Uninterruptible Power Supply) untuk server

**Catatan:** 
- Jika menggunakan cloud hosting, biaya ini **TIDAK PERLU**
- Hanya diperlukan jika sekolah memilih hosting on-premise
- Untuk cloud hosting, biaya ini bisa diabaikan

**Estimasi (jika diperlukan):**
- Server PC: Rp 3.000.000 - 4.000.000
- UPS: Rp 500.000 - 1.000.000
- Network equipment: Rp 500.000 - 1.000.000

---

### 4. Perawatan Rutin Sistem dan Dukungan Teknis (Per Tahun)
**Biaya: Rp 2.000.000 - 3.000.000/tahun**

**Cakupan:**
- Update sistem dan security patches
- Backup data rutin (harian/mingguan)
- Monitoring performa sistem
- Troubleshooting dan perbaikan bug
- Dukungan teknis untuk user (email/telepon)
- Training untuk admin sistem
- Dokumentasi update

**Layanan:**
- **Basic Support:** Rp 2.000.000/tahun
  - Support via email
  - Update security patches
  - Backup manual
  - Response time: 24-48 jam

- **Premium Support:** Rp 3.000.000/tahun
  - Support via email dan telepon
  - Update otomatis
  - Backup otomatis harian
  - Response time: 4-8 jam
  - Training 2x per tahun

---

## Rekomendasi Implementasi

### Fase 1: Development & Testing (Bulan 1-4)
- ✅ Development sistem TMS
- ✅ Testing di lingkungan lokal
- ✅ **Biaya: Rp 5.000.000 - 8.000.000**

### Fase 2: Deployment (Bulan 5)
- ✅ Setup hosting (Cloud/Web/VPS)
- ✅ Deploy aplikasi ke server
- ✅ Setup domain dan SSL
- ✅ **Biaya: Rp 1.500.000 - 3.000.000**

### Fase 3: Go Live & Maintenance (Bulan 6+)
- ✅ Launch sistem ke user
- ✅ Training untuk admin dan user
- ✅ Monitoring dan support
- ✅ **Biaya: Rp 2.000.000 - 3.000.000/tahun**

---

## Catatan Penting

1. **Biaya Development** adalah one-time cost (sekali bayar)
2. **Biaya Hosting** adalah biaya tahunan (berulang)
3. **Biaya Maintenance** adalah biaya tahunan (berulang)
4. **Biaya Perangkat Keras** hanya diperlukan jika hosting on-premise
5. **Cloud Hosting** (Railway/Render) bisa gratis untuk tier awal, cocok untuk testing
6. **Hosting Web Standar** cocok untuk production dengan budget terbatas
7. **VPS Hosting** cocok untuk production dengan traffic tinggi

---

## Kesimpulan

**Total Biaya Implementasi Awal:**
- **Minimum:** Rp 7.000.000 (dengan cloud hosting gratis)
- **Maksimum:** Rp 16.000.000 (dengan on-premise hosting)

**Total Biaya Tahunan:**
- **Minimum:** Rp 3.650.000/tahun
- **Maksimum:** Rp 6.800.000/tahun

**Rekomendasi:** Mulai dengan **Cloud Hosting (Railway/Render)** untuk testing, kemudian upgrade ke **Hosting Web Standar** atau **VPS** untuk production.
