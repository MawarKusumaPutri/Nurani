---

## Rincian Biaya Per Aktivitas

### 1. Kustomisasi dan Pengembangan Fitur Sistem TMS (Laravel)
**Estimasi: Rp 5.000.000 - 8.000.000**

Biaya ini mencakup:
- Pengembangan fitur manajemen RPP (Rencana Pelaksanaan Pembelajaran)
- Sistem presensi guru dan siswa
- Manajemen materi pembelajaran
- Sistem kalender dan pengumuman
- Manajemen surat
- Fitur data alumni
- Import/export data Excel
- Integrasi dengan berbagai modul

### 2. Hosting VPS atau Cloud untuk Deployment Sistem
**Estimasi: Rp 1.500.000 - 3.000.000**

Biaya ini mencakup:
- Biaya hosting VPS/Cloud (Railway, DigitalOcean, AWS, dll)
- Domain dan SSL certificate
- Database hosting
- Storage untuk file upload (foto, dokumen, lampiran)
- Bandwidth dan traffic

**Estimasi per tahun:**
- Hosting VPS: Rp 100.000 - 200.000/bulan
- Domain: Rp 150.000 - 300.000/tahun
- SSL Certificate: Gratis (Let's Encrypt) atau Rp 200.000/tahun

### 3. Pengadaan Perangkat Keras Tambahan (Jika Diperlukan)
**Estimasi: Rp 2.000.000 - 5.000.000**

Biaya ini mencakup (opsional):
- Komputer/laptop untuk admin TU
- Printer untuk cetak dokumen
- Scanner untuk digitalisasi dokumen
- Perangkat jaringan (router, switch)
- Webcam untuk dokumentasi kegiatan

### 4. Perawatan Rutin Sistem dan Dukungan Teknis (Per Tahun)
**Estimasi: Rp 2.000.000 - 3.000.000**

Biaya ini mencakup:
- Maintenance dan update sistem
- Bug fixing dan troubleshooting
- Backup data rutin
- Monitoring performa sistem
- Support teknis untuk pengguna
- Update keamanan dan patch

---

## Catatan Penting

1. **Biaya Kustomisasi** dapat bervariasi tergantung kompleksitas fitur yang diminta
2. **Biaya Hosting** dapat disesuaikan dengan kebutuhan traffic dan storage
3. **Perangkat Keras** bersifat opsional, tergantung kondisi existing di sekolah
4. **Perawatan Sistem** sangat penting untuk menjaga keamanan dan performa aplikasi

---

## Rekomendasi

Untuk implementasi yang optimal, disarankan:
- Menggunakan hosting cloud yang reliable (Railway, DigitalOcean, atau AWS)
- Melakukan backup data secara berkala
- Menyediakan pelatihan untuk pengguna (guru dan staf TU)
- Melakukan monitoring dan evaluasi sistem secara rutin
