============================================
CARA PAKAI - FIX FOTO PASTI MUNCUL
============================================

LANGKAH 1:
----------
Double-click: FIX_FOTO_PASTI_MUNCUL.bat

Tunggu sampai selesai.
Script akan:
- Memastikan file gambar ada di htdocs
- Memperbarui file welcome.blade.php
- Clear cache Laravel


LANGKAH 2:
----------
Setelah script selesai:

1. Buka XAMPP Control Panel
2. Klik STOP pada Apache
3. TUNGGU 10 DETIK (PENTING!)
4. Klik START pada Apache
5. Pastikan status Running (hijau)


LANGKAH 3:
----------
TEST DENGAN LOCALHOST:

1. Buka browser
2. Ketik: http://localhost/nurani/public
3. Tekan Enter
4. HARD REFRESH: Ctrl + F5 (PENTING!)
   (Ini akan clear cache browser)

Foto sekolah dan 2 anak sekarang akan muncul!


============================================
PERBAIKAN YANG DILAKUKAN:
============================================

1. Menggunakan CSS Variable untuk path gambar
2. Menambahkan JavaScript untuk memastikan gambar dimuat
3. Preload gambar untuk memastikan tersedia
4. Force re-render background image


============================================
JIKA MASIH TIDAK MUNCUL:
============================================

1. HARD REFRESH: Ctrl + F5 (PENTING!)
   Atau: Ctrl + Shift + R

2. Clear cache browser:
   - Chrome: Ctrl + Shift + Delete
   - Firefox: Ctrl + Shift + Delete
   - Edge: Ctrl + Shift + Delete
   Pilih "Cached images and files"

3. Cek file gambar ada di:
   D:\Praktikum DWBI\xampp\htdocs\nurani\public\image\foto_MTS\
   - 20251002_105433-fotor-20251022182659.png
   - 20251002_111603-fotor-20251022164553.png

4. Cek file welcome.blade.php ada di:
   D:\Praktikum DWBI\xampp\htdocs\nurani\resources\views\welcome.blade.php

5. Restart Apache lagi

6. Test dengan: http://localhost/nurani/public
   Jangan lupa: Ctrl + F5 untuk hard refresh!


============================================
TIPS:
============================================

- Selalu gunakan Ctrl + F5 setelah update file
- Pastikan Apache RUNNING (hijau)
- Tunggu 10 detik setelah STOP sebelum START
- Cek console browser (F12) untuk error
