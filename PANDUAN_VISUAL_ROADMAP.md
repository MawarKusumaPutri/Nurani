# Panduan Membuat Visual Roadmap Implementasi

## Struktur Visual Roadmap (Seperti Foto)

### Elemen Visual:
1. **Title Bar** (Bar kuning-hijau di atas)
   - Teks: "ROADMAP IMPLEMENTASI"
   - Background: Kuning-hijau
   - Teks: Putih, besar, bold

2. **Jalan (Road)** (Jalan hitam dengan garis putus-putus putih)
   - Background: Hitam
   - Garis tengah: Putih, putus-putus
   - Berkelok-kelok (winding road)

3. **Pin Lokasi** (Location Pins dengan nomor)
   - Pin 1: Pink dengan nomor "1"
   - Pin 2: Purple dengan nomor "2"
   - Pin 3: Teal/Green dengan nomor "3"
   - Pin 4: Blue dengan nomor "4" (jika ada)

4. **Kartu Tahap** (Card untuk setiap tahap)
   - Judul tahap (bold)
   - Deskripsi singkat
   - Timeline

5. **Bottom Bar** (Bar kuning-hijau di bawah)
   - Mirror dari title bar

### TAHAP 1: PERSIAPAN DAN ANALISIS

## Tools untuk Membuat Visual Roadmap

### Opsi 1: PowerPoint / Google Slides
1. Buat slide baru
2. Insert shape untuk "road" (gunakan curve tool)
3. Insert shape untuk "pins" (location pin shape)
4. Insert text box untuk setiap tahap
5. Format dengan warna sesuai

### Opsi 2: Canva
1. Pilih template "Roadmap" atau "Timeline"
2. Customize dengan data TMS
3. Download sebagai gambar

### Opsi 3: Figma / Adobe Illustrator
1. Buat artboard baru
2. Draw road dengan pen tool
3. Add location pins
4. Add text untuk setiap tahap
5. Export sebagai gambar

### Opsi 4: Online Tools
- **Lucidchart** - Timeline template
- **Miro** - Roadmap template
- **Draw.io** - Free diagram tool
