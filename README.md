# TMS Nurani - Sistem Manajemen Sekolah

Sistem manajemen sekolah berbasis web untuk MTs Nurul Aiman.

## Fitur Terbaru
- ✅ Import Data Siswa dari Excel/CSV
- ✅ Tracking Pertemuan dengan warna merah/hijau
- ✅ RPP Dropdown dengan multiple links
- ✅ Update button di card footer
