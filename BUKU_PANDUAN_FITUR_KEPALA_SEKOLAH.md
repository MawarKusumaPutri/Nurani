## 4. FITUR MONITORING GURU

### 4.1 Cara Lihat Data Guru

**Langkah-langkah:**

2. **Lihat daftar guru** dengan informasi:

## 6. FITUR APPROVAL RPP

### 6.1 Cara Lihat RPP Pending Approval

**Langkah-langkah:**

1. **Klik menu "Approval RPP"** di sidebar
2. **Lihat daftar RPP** yang menunggu approval:
   - Nama Guru
   - Mata Pelajaran
   - Pertemuan Ke-
   - Tanggal Submit
   - Status (Pending/Approved/Rejected)

### 6.2 Cara Review RPP

**Langkah-langkah:**

1. **Klik menu "Approval RPP"** di sidebar
2. **Klik tombol "Review"** pada RPP yang ingin direview
3. **Lihat detail RPP:**
   - Identitas
   - Kompetensi Inti
   - Kompetensi Dasar
   - Tujuan Pembelajaran
   - Materi Pembelajaran
   - Metode Pembelajaran
   - Kegiatan Pembelajaran
   - Penilaian
   - Pengesahan

### 6.3 Cara Approve RPP

**Langkah-langkah:**

1. **Klik menu "Approval RPP"** di sidebar
2. **Klik tombol "Review"** pada RPP
3. **Baca dan cek** semua bagian RPP
4. **Jika sudah sesuai:**
   - Klik tombol "Approve"
   - Isi catatan (optional)
   - Klik "Konfirmasi Approve"
5. **RPP status berubah** menjadi "Approved"
6. **Guru akan mendapat notifikasi**

### 6.4 Cara Reject RPP

**Langkah-langkah:**

1. **Klik menu "Approval RPP"** di sidebar
2. **Klik tombol "Review"** pada RPP
3. **Jika ada yang perlu diperbaiki:**
   - Klik tombol "Reject"
   - Isi alasan reject (required)
   - Klik "Konfirmasi Reject"
4. **RPP status berubah** menjadi "Rejected"
5. **Guru akan mendapat notifikasi** dengan alasan reject

### 6.5 Cara Bulk Approve RPP

**Langkah-langkah:**

1. **Klik menu "Approval RPP"** di sidebar
2. **Centang checkbox** pada RPP yang ingin di-approve
3. **Klik tombol "Bulk Approve"**
4. **Konfirmasi** bulk approve
5. **Semua RPP yang dicentang** akan di-approve

## 11. TROUBLESHOOTING

### 11.1 Tidak Bisa Login

**Solusi:**
1. Pastikan email dan password benar
2. Pastikan role yang dipilih adalah "Kepala Sekolah"
3. Hubungi admin jika masih tidak bisa

### 11.2 Grafik Tidak Muncul

**Solusi:**
1. Refresh halaman (F5)
2. Clear cache browser
3. Coba browser lain
4. Pastikan JavaScript enabled

### 11.3 Data Tidak Terupdate

**Solusi:**
1. Refresh halaman (F5)
2. Logout dan login kembali
3. Clear cache browser

### 11.4 Export Gagal

**Solusi:**
1. Pastikan koneksi internet stabil
2. Coba export ulang
3. Coba format lain (PDF/Excel)
